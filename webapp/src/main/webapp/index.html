Greetings from Georivals!

The following requests are available as of server version 0.5.6. This may be horribly out of date by the time you read this.

  • /download - link to mobile app .apk download
  • /battle/history - GET. Request must be made with a valid cookie sid. Returns the battle history of the player with this sid.
  • /profile - GET. Request must be made with a valid cookie sid. Returns the profile of the player with this sid.
  • /profile/id/{id} - the profile of the player with this id
  • /highscore - all highscores of all players
  • /highscore/{id} - the highscore of the player with this id
  • /movement - GET all the moving units of this player. Works only with valid sid.
  • /movement/claim?latitude=xxx&longitude=xxxx - GET. Claim the units at this location. Works only with valid sid.
  • /movement/delete?id=XX - DELETE. Cancel this movement. Works only with valid sid. NOT WORKING
  • /movement/myunits - GET all the units of the player. Works only with valid sid.
  • /movement/myunits/exclude?latitude=xxx&longitude=xxxx - GET all the units of the player that are NOT in this province. Works only with valid sid.
  • /movement/to?latitude=xxx&longitude=xxxx - POST move sent units to that location. Works only with valid sid.
  • /province - this is a POST method. Accepts JSON Strings such as { "nelongitude":5.0,"swlongitude":3.0,"swlatitude":2.0,"nelatitude":4.0 } and a Cookie "sid"
  • /province?latitude=double&longitude=double - this is a GET method. Returns the ProvinceViewDTO for this location.
  • /province/myprovinces - GET. Returns all owned provinces for this player.
  • /province/changehome?latitude=xxx&longitude=xxxx - POST. Changes the home province of the player to this location.
  • /province/rename?latitude=xxx&longitude=xxxx&newname=bla - POST. Renames this owned province.
  • /registration/phase1 - this is a POST method. Accepts JSON Strings as RegistrationDTO and outputs RegistrationResponse
  • /registration/phase1?userName=XXXX - this is a GET method. Outputs RegistrationResponse
  • /registration/phase2 - this is a POST method. Accepts JSON Strings as RegistrationDTO and outputs RegistrationResponse
